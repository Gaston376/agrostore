# AgroStore - Quality Agricultural Products

A simple, responsive website for an agricultural products store.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Product Catalog**: Display of various agricultural products
- **Shopping Cart**: Add products to cart functionality
- **Contact Form**: Easy way for customers to reach out
- **Smooth Animations**: Modern UI with smooth scrolling and transitions

## Products Offered

1. **Seeds** - High-quality seeds for various crops
2. **Fertilizers** - Organic and chemical fertilizers
3. **Farm Tools** - Essential farming equipment
4. **Irrigation** - Water management systems
5. **Animal Feed** - Nutritious feed for livestock
6. **Pesticides** - Safe and effective pest control

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Responsive Design

## How to Use

1. Open `index.html` in a web browser
2. Browse products in the catalog
3. Click "Add to Cart" to add items
4. Fill out the contact form to get in touch
5. Navigate using the header menu

## Deployment

This is a static website that can be deployed to:
- Vercel
- Netlify
- GitHub Pages
- Any static hosting service

Simply upload all files maintaining the folder structure.

## Customization

- **Colors**: Edit the CSS variables in `style.css`
- **Products**: Modify the product cards in `index.html`
- **Contact Info**: Update phone, email, and address in the contact section
- **Logo**: Replace `images/logo.png` with your own logo
